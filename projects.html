raspizza
Transfer learning network from EfficientNetB0, for recognising Pizzas' baking level from images. The network was quantized and executed in a Raspberry Pi Zero 2. This project is a personal project in collaboration with university, and originated from a pizza hoven that I have modded and wanted to automatize.
C++, Python3, Makefile.
